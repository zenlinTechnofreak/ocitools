# NAME
ocitools-validate - Validate an OCI bundle

# SYNOPSIS
**ocitools validate**  *[OPTIONS]*

# DESCRIPTION

Validate an OCI bundle

# OPTIONS
**--help**
  Print usage statement

**--path=PATH
  Path to bundle

**--hooks**
  Check specified hooks exist and are executable on the host.
